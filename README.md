# tinter-weather-appp

A simple weather application built with Python's Tkinter for the GUI and OpenWeatherMap's API for weather data.

## Features

- Displays current weather for a given city.
- Shows temperature, humidity, pressure, and weather description.
- Built using Tkinter for the graphical user interface.

## Requirements

- Python 3.x
- Tkinter (should come pre-installed with Python)
- `requests` library (can be installed using `pip`)

## Installation

1. Clone this repository to your local machine:

   ```bash
   git clone https://github.com/your-username/tkinter-weather-app.git
   cd tkinter-weather-app
   
2.Install the required dependencies:
    pip install -r requirements.txt

## Usage

1. Open the app.py file in a text editor or IDE.
2. Run the application:
     python app.py
3. Enter a city name, and the app will display the current weather.
